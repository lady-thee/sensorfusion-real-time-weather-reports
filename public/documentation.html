API Documentation

Welcome to the Sensor API Documentation. This guide provides comprehensive instructions on utilizing the Sensor API efficiently. To begin, sign up as an Operator to gain access to the API functionalities.

How to Use

Once registered as an Operator, navigate to the dashboard to register a remote sensor or system. The registration process requires minimal sensor details, predominantly the sensor's name, location etc.

API Link

Access the Sensor API via the following link: http://localhost:8000/api/sensor/api
